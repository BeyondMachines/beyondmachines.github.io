Challenge Rules

How to Participate
  1. Start from this page - it contains your first clue
  2. Find and collect tokens throughout the challenge
  3. Each token leads to the next step
  4. Document your journey in a detailed writeup
  5. You can submit even if you haven't collected all tokens
  6. Get unstuck: Daily writeups of general techniques start December 16th on our Discord server - learn without spoilers!
Important Guidelines
  • No destructive actions: Do not attempt to damage, deface, or disrupt any systems
  • Writeup required: Detailed documentation is essential for prize consideration
  • Be ethical: Follow responsible disclosure practices
  • Ask for help: Join our Discord if you're stuck. We'll give hints if people are desperately stuck, but no spoilers!
  • Have fun: This is a learning experience - enjoy the journey!
